Google Chrome for mobile introduces five new features:

1. **Chrome Actions**: Quick options in search results, such as calling a restaurant directly.
2. **Shortcut Suggestions**: Identifies sites based on frequent queries, for example, suggesting a subway timetable site when you type "timetables".
3. **Trending Searches**: Displays popular search queries when you tap the address bar.
4. **Enhanced Address Bar**: Features a redesigned interface for tablets, which overlays the website.
5. **Live Sports Updates**: Provides real-time updates on games of followed teams within the Discover Feed.

These enhancements aim to simplify mobile browsing, making tasks and information more readily available.
