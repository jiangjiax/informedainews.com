Front-end developer spent 9 months developing a low-code platform tailored for middle and back-office management systems. Users can dynamically create projects, build pages, and implement functionalities without coding.

Features:
- Complete project creation, including name, logo, and menu.
- Supports RBAC (Role-Based Access Control) permission configuration and page permission control.
- Logic orchestration through event flows.
- Interface configuration supports parameter handling.
- Multi-environment deployment and rollback.
- Custom component development.

Links:
- Online address: http://www.marsview.cc/
- Open-source address: https://github.com/JackySoft/marsview
- Documentation address: http://docs.marsview.cc/project/

RBAC: Role-Based Access Control, a model for managing permissions based on roles, simplifying access management.
