## DJI的平价驾驶辅助技术套件将革新汽车行业

<img
  src="https://image.informedainews.com/2024/06/06/AI-l39-1717673902-0.jpg"
  alt="DJI的平价驾驶辅助技术套件将革新汽车行业"
/>

DJI Automotive，无人机巨头DJI的衍生公司，正在革新汽车驾驶员辅助技术的可负担性。通过利用无人机技术，他们以每辆车仅5000元人民币的成本提供Level 2自动驾驶功能，使得这项技术即使在售价约100000元人民币的汽车中也变得可行。这与竞争对手如华为形成鲜明对比，后者的系统通常出现在更昂贵的车型中。

今年至少有20款新车型将采用DJI的技术，该技术使用摄像头和廉价的半导体，而不是昂贵的激光雷达和雷达系统。这种方法借鉴了DJI Phantom 4无人机技术，使车辆能够根据摄像头输入控制方向和速度。

DJI Automotive技术的吸引力在于其成本效益，可能加速驾驶员辅助系统的普及。中国高度重视此类技术，目标是到2030年，Level 2和Level 3车辆占新车销量的70%，Level 4车辆占20%。

DJI Automotive计划在今年年底前推出一款更先进的系统，能够在常规道路上实现自动驾驶，每辆车定价为7000元人民币。这一举措可能会进一步普及自动驾驶技术，使其面向更广泛的市场。

Level 2自动驾驶指的是系统在某些条件下能够控制转向和加速/减速，但要求驾驶员监控环境并准备随时接管。Level 4自动驾驶允许车辆在特定条件下处理所有驾驶方面，无需任何人为干预。

智能技术与车辆的融合正在加速。据预测，到2025年，中国75.9%的新车将配备内置互联网连接。这一趋势在股市中得到了体现，如宏泉物联和中海达等公司股价大幅上涨。

政府正积极推动这一转变。工业和信息化部及其他机构已批准九个联盟进行试点项目，旨在完善智能汽车的规定和标准。此外，还专门分配了1亿个电话号码，用于未来的车对万物（C-V2X）通信。

向智能互联车辆转型不仅仅是便利性的提升，它标志着我们与交通互动方式的根本变革。随着这些技术的成熟，它们有望带来更安全、更高效的旅行，并可能重塑汽车行业的格局。

**术语解释：**

- **智能或互联车辆：** 利用互联网连接和车载传感器来增强安全性、导航和其他功能的汽车。
- **C-V2X（蜂窝车对万物）：** 使车辆能够与其他车辆（V2V）、基础设施（V2I）、行人（V2P）和网络（V2N）进行通信的技术。
- **L3自动驾驶：** 一种自动驾驶级别，车辆可以处理大多数驾驶任务，但在某些情况下需要人为干预。

### Ubuntu Core 24 发布：针对物联网和嵌入式设备优化

<img
  src="https://image.informedainews.com/2024/06/06/Ar-KjD-1717673952-0.jpg"
  alt="Ubuntu Core 24 发布：针对物联网和嵌入式设备优化"
/>

Canonical 发布了 Ubuntu Core 24，这是针对物联网和嵌入式设备定制的 Ubuntu 24.04 LTS 完全容器化版本。该版本支持广泛的硬件，包括 Raspberry Pi、Qualcomm DragonBoard、Intel NUC，以及基于 x86_64 和 RISC-V 架构的设备。

Ubuntu Core 24 引入了完全可抢占的内核，增强了响应能力，并在认证硬件上实现了先进的实时功能。关键特性包括可定制的图像验证集、对自动网络请求的增强控制，以及为 AIoT 设备和图形应用提供的改进的 GPU 集成。它还支持 Ubuntu Frame 以增强嵌入式 Linux 安全性，并集成 Canonical Landscape 进行设备管理。

此版本与 Azure IoT Edge 无缝集成，并支持机器人操作系统（ROS）。Ubuntu Core 24 本质上是一个为下一代智能设备，特别是 AIoT 和边缘计算领域设计的强化嵌入式 Ubuntu 版本。它提供了一个完整的 AI 堆栈，包括实时内核，这对于在工业、汽车和安全敏感的嵌入式解决方案中做出关键决策至关重要。

最低系统要求为 384MB RAM 和 512MB 存储，对于 UEFI 安全启动和 FED 则需要更高的要求。该版本支持众多设备，确保了广泛的应用性。
