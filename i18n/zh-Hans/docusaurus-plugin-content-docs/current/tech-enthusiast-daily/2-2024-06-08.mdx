### 在iPhone上进行滚动截图的指南

在iPhone上截取屏幕截图非常简单，只需快速按下音量增大和侧边按钮即可。对于更长的截图，即滚动截图，过程稍微复杂一些。这些截图允许你一次性捕捉整个网页、文档或电子邮件，而不是将多个图像拼接在一起。

创建滚动截图的方法如下：
1. 使用侧边和音量增大按钮（或侧边/顶部和主页按钮，适用于带有Touch ID的型号）截取常规屏幕截图。
2. 快速点击屏幕左下角出现的预览。
3. 选择“整页”以查看整个捕捉的预览。
4. 如有必要，进行编辑，然后选择“完成”。
5. 保存为PDF到“文件”中，或者如果可用，保存到“照片”中。

长截图通常作为PDF保存在“文件”应用中，可以在那里查看、编辑和分享。如果截图的尺寸适合屏幕，它也可能直接保存到“照片”中。

这一功能简化了长内容的捕捉，增强了iPhone在文档记录和分享方面的实用性。

### 中国启动大规模交通现代化计划

中国交通运输部联合十二个其他部门，启动了一项全面计划，旨在现代化交通基础设施。该计划名为“交通大型装备更新行动计划”，涵盖了七个关键领域：

1. **公共汽车电气化** - 转向电动公交车以减少排放。
2. **老旧柴油卡车替换** - 逐步淘汰老旧、污染车辆。
3. **船舶和机车的报废与更新** - 替换过时的海运和铁路运输工具。
4. **邮政和快递设备的替换** - 升级邮政和配送服务的技术。
5. **物流设施的升级** - 提高物流网络的效率和可持续性。
6. **设备标准的提升** - 提高设备质量和性能的标准。

目标明确：转向更清洁、更高效、碳排放更少的交通系统。这一举措强调了对环境可持续性和行业现代化的更广泛承诺。

**洞察：**
该计划不仅仅是更新机械设备；它是一个向绿色经济战略转变的标志。通过针对公共交通和物流等高排放领域，中国为全球环境管理树立了先例。特别是向电动车辆的过渡和物流效率的提升，因其直接影响公众健康和经济生产力而尤为值得注意。这一倡议反映了对气候变化的积极应对，与全球减少碳足迹的努力相一致。

2024年5月：史上最热

2024年5月，地球经历了有记录以来最热的5月。哥白尼气候变化服务机构（C3S）的数据揭示，全球平均气温比1991-2020年的平均值高出0.65°C，比工业化前水平高出1.52°C。

连续破纪录

这是全球气温连续第12个月创下相应月份的记录。过去12个月，全球平均气温比1991-2020年的平均值高出0.75°C，比1850-1900年的工业化前水平高出1.63°C。

气候警钟

联合国秘书长引用C3S数据，呼吁紧急行动。世界气象组织报告警告，我们正逼近《巴黎协定》的气候临界值。预计2024年至2028年，全球平均气温将比工业化前高出1.1°C至1.9°C。

行动的紧迫性

全球变暖速度达到历史新高，每十年增加0.26°C。剩余的碳预算，即在导致全球升温1.5°C之前可排放的二氧化碳量，仅剩约200千兆吨，相当于五年排放量。

总结

气候变化的数据如同烈日下的阴影，清晰而无情。我们正站在历史的转折点，每一度的升温都是对未来的挑战。现在是行动的时刻，否则我们将无法逆转这一趋势。

### 淘宝天猫更新评分体系，强化店铺体验分影响搜索排序

<img
  src="https://image.informedainews.com/2024/06/08/Ar-dkA-1717825137-1.jpg"
  alt="淘宝天猫更新评分体系，强化店铺体验分影响搜索排序"
/>

淘宝天猫近期推行新评分体系，以“店铺体验分”和“商品体验指数”取代原有的DSR评价体系。这一变革直接影响搜索排序和店铺流量。体验分高的店铺将在搜索结果中更靠前，从而吸引更多流量。淘宝天猫强调用户体验，鼓励商家提升服务质量。商家可通过“千牛后台”查看体验分，并在商品详情页展示相关服务指标，如发货速度和客服响应时间，直接影响消费者决策。

**解释：**

- **DSR（Detailed Seller Ratings）：** 淘宝此前的卖家服务评价体系，包括商品描述相符、卖家服务态度、物流服务质量等评分。
- **店铺体验分：** 新评价体系的一部分，综合考量店铺服务质量，直接影响店铺在平台上的曝光率和流量。
- **商品体验指数：** 衡量商品服务质量的指标，包括发货速度、客服响应等，直接影响消费者购买决策。
- **千牛后台：** 淘宝商家管理工具，商家可在此查看和管理店铺数据。

**洞察：**

淘宝天猫的这一策略调整，标志着电商平台从单纯的价格竞争转向服务质量的竞争。通过提升用户体验，商家不仅能增强顾客忠诚度，还能在竞争激烈的市场中脱颖而出。这一转变对商家而言既是挑战也是机遇，需要他们重新审视和优化服务流程，以满足日益增长的消费者期望。

### 职场成功战略沟通技巧

**文章焦点：** 如何有效地向上级沟通工作贡献。

**关键点：**
1. **主动汇报：** 定期更新上级关于项目里程碑和挑战的情况。使用具体数据和对比来突出成就。
2. **策略性抱怨：** 承认工作量而不夸大，强调解决方案和效率。
3. **结构化演示：** 使用清晰格式（事件、数据、对比）来展示复杂任务及其结果。
4. **问题解决方法：** 在提出问题时，包含深思熟虑的解决方案和考虑。

**个人反思：**
本文的核心在于职场中的沟通艺术。它不仅仅是展示努力，而是通过策略性汇报和问题解决来展示价值。特别强调使用数据和清晰对比非常有效，因为它将抽象的努力转化为具体的贡献。然而，这种做法应基于真正的生产力和诚信。对工作量的过度夸大或操纵可能会损害信任和信誉。

**术语解释：**
- **向上汇报：** 与高层管理分享工作进展和挑战的过程。
- **工作量：** 分配给个人或团队的工作量。
- **数据驱动：** 基于事实数据而非假设做出决策或提供信息。
- **问题解决：** 寻找解决问题的有效方法的过程。
