## 克什米尔邦举行十年来的首次议会选举

印度北部的查谟和克什米尔地区于2024年9月18日举行了十年来的首次议会选举。此次选举是在总理纳伦德拉·莫迪领导的印度人民党于2019年废除该地区特殊地位之后进行的。这一特殊地位是查谟和克什米尔独有的，仅允许当地居民购买房产和获得工作。长期以来，该地区一直是印度和巴基斯坦之间的冲突热点，其自治权被取消，并被划分为两个联邦直辖区：查谟和克什米尔，以及拉达克。

此次选举旨在组建一个推动地区发展和安全的当地政府。一些政党寻求恢复该地区的邦地位。在90个席位和870万注册选民的参与下，此次选举至关重要。第二和第三阶段的投票分别定于9月25日和10月1日进行，结果将于10月8日公布。

莫迪在一次集会上承诺将查谟和克什米尔建设成“安全繁荣”的地区，并批评过去的政府忽视了人民的需求。自自治权被废除以来，印度人民党在该地区的受欢迎程度有所下降，一些观察人士指出其在克什米尔地区的弱势存在。

此次选举意义重大，反映了该地区持续的紧张局势和复杂的动态，这一地区对印度的身份和安全至关重要。

### 中国渐进式延迟退休年龄：一种平衡的方法

<img
  src="https://image.informedainews.com/2024/09/Soc-BWf-1726657871-0.jpg"
  alt="中国渐进式延迟退休年龄：一种平衡的方法"
/>

中国最近决定提高退休年龄并非一时冲动，而是经过深思熟虑的举措。男性将在63岁退休，女性在58岁退休。这些变化将在15年内逐步实施，影响特定年份之后出生的人群。

这一策略非常巧妙。1968年出生的人，即高出生率年份的人，将多工作近一年。女性，尤其是面临困难的蓝领工人，将获得更长的退休时间来享受家庭生活。

这不仅仅是数字问题，而是平衡经济需求与社会公平的问题。这些变化有利于在邓小平改革后开始工作的人群。它承认了老一代人的奋斗，同时为更年轻、受过更好教育的工人铺平了道路。

对于年轻一代来说，延迟退休不是负担，而是机会。许多人，尤其是受过教育的城市女性，可能更愿意延长工作时间。

本质上，这项政策是实用主义与同情的结合。它延长了工作年限以维持经济增长，同时确保老年工人有更多时间享受晚年生活。

### 尼日利亚洪水导致越狱和大规模流离失所

<img
  src="https://image.informedainews.com/2024/09/Soc-IkG-1726657904-0.jpg"
  alt="尼日利亚洪水导致越狱和大规模流离失所"
/>

在尼日利亚迈杜古里，超过200名囚犯从被洪水淹没的监狱中逃脱。这场洪水是20年来最严重的，紧随9月9日大坝倒塌之后。37人死亡，20万人流离失所。国家紧急事务管理局（NEMA）报告称，建筑物被淹没，局势一片混乱。

尼日利亚惩教署（NCS）表示，在转移至更安全设施的过程中，281名囚犯逃脱，其中7人被重新抓获，仍有274人在逃。洪水摧毁了监狱围墙和工作人员宿舍。目前正在进行与其他安全机构的联合搜查。

来自Sanda Kyarimi公园动物园的野生动物，包括爬行动物和狮子，也被迫进入居民区。六个营地的临时避难所安置了流离失所者。迈杜古里曾是博科圣地叛乱的中心，近年来关闭了之前的国内流离失所者营地。

联合国粮食及农业组织（FAO）警告称，3180万尼日利亚人面临严重的粮食不安全问题。援助人员担心，在拥挤的营地中，情况会恶化，并可能爆发霍乱等疾病。

今年4月，尼日利亚水文服务局预测全国36个州将发生洪水。由于气候变化，实际影响超出了预测。29个州受到影响，130万公顷（320万英亩）土地被淹没，其中一半是农田。

副总统卡西姆·谢蒂马承认洪水的严重性，并誓言要找到持久的解决方案。然而，基础设施难以应对。尼日利亚行动援助组织的苏瓦伊巴·丹卡博指出，被毁的道路网络使援助物资的运送变得复杂。

这种情况与喀麦隆远北地区的洪水相似，那里的堤坝破裂，导致3700所房屋倒塌。贝努埃河是喀麦隆和尼日利亚共有的河流，承载着每年引发洪水的拉戈大坝。像阿达马瓦这样的州，毗邻博尔诺州，面临着预期大坝排放带来的迫在眉睫的风险。

### 中国企业突破OLED屏幕核心技术垄断，助力产业升级

中国OLED行业历经十年追赶，2024年首季产量超越韩国，跃居全球第一。OLED技术以其高对比度、超薄、可弯曲等优势，成为新一代显示技术。然而，核心材料FMM（精密金属掩模版）长期被日本DNP垄断，制约中国OLED产业发展。

徐华伟创立的「众凌科技」，专注FMM国产化，历经四年研发，突破因瓦合金材料瓶颈，实现FMM量产。2023年，「众凌科技」FMM产品通过客户验证，打破DNP垄断，降低国内面板厂对单一供应商的依赖。

FMM是OLED生产关键模具，直接影响面板品质。DNP对不同客户实行品质代差，限制中国面板厂高端产品制造能力。「众凌科技」通过自主研发因瓦合金，提升FMM品质，已应用于各类终端品牌中高端产品。

未来，随着「众凌科技」等企业崛起，中国OLED产业将摆脱技术封锁，迎来更广阔发展空间。

东南亚的QR支付正在蓬勃发展，尤其是在马来西亚和柬埔寨。这些国家由于银行账户拥有率低、ATM机数量有限以及智能手机价格实惠，成为了QR支付的领先者。

在柬埔寨，2023年QR支付交易量激增29%，达到6.01亿笔。柬埔寨国家银行推出了Bakong Tourists，这是一个面向旅行者的移动支付系统，推广使用柬埔寨瑞尔。全国有330万个QR支付地点，即使是小商贩也接受QR码支付。

马来西亚的DuitNow QR于2019年推出，2024年上半年交易量达到15亿笔，总额为137亿林吉特。它与其他东盟系统互通，使跨境支付更加便捷。

新加坡的PayNow于2017年推出，2023年处理了4.37亿笔交易，总价值为1570亿新加坡元。2021年，PayNow与泰国的PromptPay连接，成为首个国际即时支付系统链接。PayNow现在支持初创企业和金融科技公司，有27家参与银行和公司。

泰国银行正在扩大PromptPay在整个东盟地区的覆盖范围。QR支付正在成为促进商业和旅游业发展的基础设施。

### 微软Copilot：借助AI工具提升生产力

Microsoft Copilot 是一套 AI 工具，旨在提升各种 Microsoft 软件和服务中的生产力。它包括十几个以 Copilot 品牌命名的产品，每个产品都针对特定任务进行了定制，例如在 Outlook 中总结电子邮件或在 GitHub Copilot 中生成代码。

Copilot 与 Windows、Bing 和 Microsoft Edge 集成，提供写作、翻译和图像创建等功能。它还可以控制某些 Windows 11 设置，充当数字助理。

主要有两个版本：Copilot Pro（每月 20 美元）和 Microsoft 365 Copilot（每位用户每月 30 美元）。Pro 版本提供对 Microsoft 365 应用中高级 AI 模型和功能的优先访问权限，而 365 Copilot 则包括企业级数据保护和个性化响应。

Copilot Studio 允许用户自定义 Copilot，为特定任务创建定制的 AI 助手。GitHub Copilot 是独立于主套件的，帮助开发人员进行代码生成和故障排除。

Microsoft 的 Copilot 生态系统庞大，旨在增强生产力并简化各种平台和用户需求中的任务。

### NASA授予Intuitive Machines 48亿美元合同用于月球通信基础设施

NASA 计划在月球上建立永久性人类存在。目前，宇航员只能在视线直接对准地球时与地球通信。为了克服这一问题，NASA 正在开发近地空间网络，并向 Intuitive Machines 授予了一份价值高达 48.2 亿美元的合同，用于建造月球通信卫星星座。

总部位于休斯顿的 Intuitive Machines 以其二月份的私人登月任务而闻名，尽管着陆过程并不顺利。该公司提供前往月球的共享服务，但由于需要与地球保持直接视线，数据传输面临限制。

这份合同保证金额为 1.5 亿美元，可能延长至十年。这一基础设施对于 NASA 计划在本十年末将人类送上月球南极至关重要，因为在那里与地球的直接通信受到限制。除了载人任务外，中继系统还将增强无人任务，实现远程操控月球车和改进数据传输。

这一基础设施不仅将惠及 NASA，还将惠及包括 Intuitive Machines 在内的私营行业。该公司去年上市，严重依赖政府合同，今年第二季度收入增长了 130%，达到 4140 万美元。
