### 波兰启动“与军队共度假”活动以加强国防

### 研究提议对亿万富翁财富征收全球可行税

法国经济学家加布里埃尔·祖克曼最近的一项研究指出，对全球3000位亿万富翁的财富征收国际税在技术上是可行的，每年可能产生高达2500亿美元的收入。祖克曼受巴西G20轮值主席国委托，认为对跨国公司征税的进步为全球个人财富税铺平了道路，即使没有普遍参与。该提案建议对亿万富翁财富征收2%的年税，旨在解决当前对他们财富的0.3%的低税率问题，这远低于普通工人的税率。祖克曼强调了评估亿万富翁财富的简单性，这些财富主要以股票形式持有，并提议通过加强出境税等机制在全球范围内实施该税。该计划将在即将于里约热内卢举行的G20财长会议上讨论。

祖克曼的提案强调了全球税收规范的转变，专注于财富集中而非仅仅是收入。尽管存在政治复杂性，这种税收的可行性反映了人们对公平财富分配需求日益增长的共识。利用现有税收执行机制（如出境税）的策略显示了实施的实用性。如果这一倡议被采纳，它可能会重塑全球经济格局，不仅影响税收政策，还影响超级富豪的财富管理策略。

### 天空细胞公司获得5900万美元投资，用于可持续药品运输解决方案

瑞士初创企业SkyCell在D轮融资中筹集了5900万美元，使其总融资额达到1.16亿美元。这笔资金将支持其在亚洲和美国的扩张，重点是智能容器在药品运输中的应用。这些容器利用机器学习技术，为温度敏感药物维持最佳条件，减少损耗和排放。

SkyCell成立于2012年，最初设计存储设施，后转型为制药巨头提供运输解决方案。公司增长强劲，年增长率达50%，每月处理价值25亿美元的药品。其容器更轻便高效，能减少50%的二氧化碳排放。

全球推动脱碳化与SkyCell的使命相契合，因为制药公司正寻求减少其对全球4.4%排放量的贡献。首席执行官Richard Ettl指出，绿色转型可能耗费制药公司约2%的销售额，但高效运输带来的节省是显著的。

SkyCell的技术不仅限于运输，其智能温度计等组件在药品存储环境中也得到应用。公司的集成硬件和软件解决方案使其成为制药物流领域的关键参与者，满足了安全可持续药品配送的迫切需求。

### k-ID 获得 4500 万美元投资，以加强游戏中的儿童安全合规性

k-ID 获得 4500 万美元投资，旨在简化游戏开发者的儿童安全合规流程。这家由 Andreessen Horowitz 和 Lightspeed Venture Partners 支持的初创公司，致力于简化对不断变化的全球法规的遵守。

游戏开发者面临一项艰巨任务：在应对复杂的儿童安全法律的同时，创造吸引人的游戏。k-ID 的平台通过 API 或 SDK 提供，自动化合规流程，调整游戏功能以满足当地要求。它提供儿童单点登录、通过 QR 码实现的家长控制，以及根据年龄和地区定制的访问权限。

这笔资金凸显了游戏行业中的一个关键需求，因为开发者越来越多地瞄准全球更年轻的受众。k-ID 的解决方案不仅减轻了监管负担，还增强了家长监督，确保游戏既有趣又安全。

与 ESRB 的隐私认证 Kids Seal 合作进一步证明了 k-ID 对游戏隐私和安全的承诺。随着法规的不断演变，像 k-ID 这样的工具对于努力在创意与合规之间取得平衡的开发者来说至关重要。

### Meta的Threads扩展至Fediverse：社交平台互操作性的新时代

Meta的Threads社交平台现已扩展其覆盖范围至fediverse——一个使用ActivityPub协议相互连接的服务器网络。该功能最初在美国、加拿大和日本进行beta测试，现已面向18岁以上的Threads用户在超过100个国家开放。

Meta旨在解开fediverse的神秘面纱，教育用户了解其机制和益处。Threads用户现在可以在Mastodon等平台上分享内容，查看来自fediverse的点赞和回复，尽管直接回复功能尚未启用。

此beta阶段仅允许顶级帖子和自回复进行联邦化，排除了其他帖子类型如投票。要激活此功能，用户必须在账户设置中启用“Fediverse分享（Beta）”。

Meta的策略强调了对在fediverse内培育一个安全、多元且可互操作的社区的承诺，旨在增强内容传播和社区参与度。随着Threads已拥有超过1500万月活跃用户，这一整合可能会显著扩大fediverse的用户基础。

### 科学家工程化植物以生产关键的婴儿营养成分

### 中国的榴莲热潮正在改变东南亚经济

<img
  src="https://image.informedainews.com/2024/06/Foo-oXm-1719367999-3.jpg"
  alt="中国的榴莲热潮正在改变东南亚经济"
/>

榴莲，曾经是东南亚的一种廉价水果，如今却价值连城。中国对这种气味浓烈的水果的喜爱已经改变了当地经济。出口到中国的榴莲从2017年的5.5亿美元激增至去年的67亿美元。

前卫星编码员Eric Chan通过销售榴莲酱获利，他的初始投资增长了近50倍。曾经贫穷的农民现在重建家园，送孩子去海外大学。

泰国在榴莲出口方面领先，马来西亚和越南紧随其后。泰国的榴莲种植面积翻了一番，马来西亚的丛林被清理出来用于新的种植园。

中国投资延伸到泰国的榴莲物流，控制了约70%的批发市场。这一变化让当地泰国企业感到担忧。

榴莲，价格昂贵且是当地文化的重要组成部分，面临一个悖论：出口繁荣但价格上涨让当地消费者感到沮丧。这种曾经丰富的水果现在已成为奢侈品。

由于榴莲强烈的气味，将其运输到中国面临挑战，导致出现了冷冻后再运输等创新解决方案。
