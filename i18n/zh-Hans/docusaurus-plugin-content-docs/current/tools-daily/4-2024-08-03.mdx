### PocketBase：一个单文件后端解决方案

PocketBase，一个开源的单文件后端，于2022年7月发布。它按目录结构组织，如apis和core，并在README.md中提供安装指南。通过GitHub Actions的支持，它高效地简化了后端任务。

开源：软件，其源代码任何人都可以检查、修改和增强。
GitHub Actions：GitHub仓库内的自动化工具，用于测试和部署等任务。

### DeepSeek API：通过基于磁盘的上下文缓存实现成本降低90%的高效方案

DeepSeek API 引入了基于磁盘的上下文缓存功能。无需代码调整，该功能将常用数据存储在分布式存储中，通过这种方式，API 成本降低了 90%。非常适合大型提示，例如频繁引用的 128K 大小的提示。

**上下文缓存：** 存储重复使用的数据以加快访问速度并降低成本。

**分布式存储：** 一种系统，其中数据分布在多台计算机上，增强了可靠性和访问速度。

<Link href="https://x.com/deepseek_ai/status/1819358570766643223" color="#B0C4DE">Source</Link>

### 鸭浮：macOS多功能浮动工具

鸭浮（Duck Float）悬浮在macOS全屏应用上方，提供文件浏览、文本存储、剪贴板管理和网页浏览功能。它通过快捷方式简化多任务处理，并适应多显示器环境。

特点：
- 文件预览和拖放功能。
- 文本和链接存储。
- 带过滤和搜索功能的剪贴板历史记录。
- 在任何全屏应用内进行网页浏览。

从App Store或官方网站下载。

**解释：**
- **macOS：** 苹果公司为Mac电脑开发的操作系统。
- **剪贴板管理：** 管理复制的文本或项目，以便于粘贴。
