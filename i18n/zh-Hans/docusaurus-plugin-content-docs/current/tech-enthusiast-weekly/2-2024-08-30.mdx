# 科技迷周刊(2024-08-30) : 自动化通配符SSL证书管理与GitHub Actions

## 自动化通配符SSL证书管理与GitHub Actions

GitHub Actions 自动化管理 SSL 证书。它处理通配符证书，覆盖所有子域名。证书通过 git 提交存储。每日检查即将过期的证书，并在到期前 30 天内触发续订。报告更新一个 markdown 文件。该系统保障 ECDSA 和 RSA 证书的安全。

通配符 SSL 证书保护一个域名的所有子域名。ECDSA 和 RSA 是用于安全数据传输的加密算法。

### 科学家在高温度超导领域取得突破

<img
  src="https://image.informedainews.com/2024/08/30/Te-jY1-1725015147-0.jpg"
  alt="科学家在高温度超导领域取得突破"

科学家在高温超导领域取得重大突破，开发出一种新型超导导线，电流密度达到每平方厘米1.9亿安培。这种导线基于钡铜氧化物和稀土元素制成，成本可控，工作温度范围广，从4.2K到77K。

该导线采用离子束辅助沉积氧化镁衬底技术，通过相分离和应变驱动自组装工艺制备。纳米级缺陷的引入显著提高了电流承载能力。在4.2K和20K条件下，7T磁场强度下，钉扎力分别达到6.4TN/m3和4.2TN/m3。

这项技术有望在能源、电力、交通、医疗等领域带来变革。特别是在商业核聚变领域，HTS导线具有巨大潜力。全球已有超过40家公司投入数十亿美元开发HTS导线。

研究者戈亚尔强调，提高性价比是实现超导体大规模应用的关键。这项研究结果将指导行业进一步优化制造条件，提升商用涂层导体的性价比。

### NASA的冷原子实验室推进量子空间科学

<img
  src="https://image.informedainews.com/2024/08/30/Te-udy-1725015156-0.jpg"
  alt="NASA的冷原子实验室推进量子空间科学"
/>

<img
  src="https://image.informedainews.com/2024/08/30/Te-qod-1725015156-1.jpg"
  alt="NASA的冷原子实验室推进量子空间科学"
/>

国际空间站上的NASA冷原子实验室利用量子技术深入探索太空科学。它探测引力、暗物质和暗能量，并在微重力环境下测试爱因斯坦的相对论。

该实验室是国际空间站上的首个此类设施，使用一种称为原子干涉仪的工具。这种设备以高精度测量引力及其他力，对于理解引力的基本原理和改进导航技术至关重要。

在太空中，这些测量可以揭示行星和卫星的组成，还能追踪地球上水与冰的移动。实验室的发现可能有助于阐明宇宙中的神秘力量——暗物质和暗能量。

冷原子实验室将原子冷却至接近绝对零度，创造出一种称为玻色-爱因斯坦凝聚态的状态。这种状态使得原子在大尺度上展现出量子行为，从而更易于研究。

在微重力环境下，这些量子状态持续时间更长，为科学家提供了更多研究时间。实验室的工作可能引领新的量子技术发展，并加深对宇宙基本粒子的理解。

### DNA技术在长期数据存储与计算中的应用

<img
  src="https://image.informedainews.com/2024/08/30/Te-3wM-1725015156-0.jpg"
  alt="DNA技术在长期数据存储与计算中的应用"

<img
  src="https://image.informedainews.com/2024/08/30/Te-EUZ-1725015156-1.jpg"
  alt="DNA技术在长期数据存储与计算中的应用"

<img
  src="https://image.informedainews.com/2024/08/30/Te-sVA-1725015156-2.jpg"
  alt="DNA技术在长期数据存储与计算中的应用"
/>

<img
  src="https://image.informedainews.com/2024/08/30/Te-i1N-1725015156-3.jpg"
  alt="DNA技术在长期数据存储与计算中的应用"
/>

科学家们开发了一种在DNA中存储数据的方法，有望实现数千到数百万年的存储。这项技术使用了一种柔软的聚合物，即DNA的支架，可以干燥以进行长期存储，并在需要时重新水化以检索数据。这种聚合物称为树枝状聚合物，形成了一个纳米级纤维网络，实现了高密度数据存储——大约每立方厘米10千兆字节。

这种方法通过多次干燥和重新水化循环保持数据完整，远超过裸露的DNA。它非常适合长期存档，DNA的半衰期在4°C时为6,000年，在-18°C时为200万年。

将数据转换为DNA的ACGT代码并通过RNA测序检索意味着在读取过程中不会对DNA造成损害。该系统还直接在DNA中进行计算，通过解决简单的国际象棋和数独谜题进行了演示。

这一突破不仅推动了DNA数据存储的发展，还为DNA计算铺平了道路。
