## 特斯拉上海工厂：本地化与效率的展示

<img
  src="https://image.informedainews.com/2024/06/06/Ar-wYl-1717676014-0.jpg"
  alt="特斯拉上海工厂：本地化与效率的展示"
/>

特斯拉上海工厂，现代制造业的奇迹，已深度融入中国供应链。短短四年间，其95%的零部件实现本地采购，与400多家国内供应商合作。这一迅速的本土化进程彰显了中国强大的工业实力以及特斯拉对其的战略依赖。

自2019年投产以来，该工厂已成为全球生产效率的典范，实现了惊人的30秒整车下线周期。其创新的双层建筑设计和机器人自动化技术——车身仿佛在空中飞舞——极大提升了空间利用和生产效率。工厂内6000吨级的冲压机可一次性成型整个车身部件，简化了装配流程，增强了结构完整性。

自动化在这里无处不在，95%的生产流程由机器控制。在焊接区域，数百台机械臂取代了数百名工人，确保了更快速、更统一的生产标准。

此工厂不仅是特斯拉的主要出口基地，2023年占其全球产量的一半以上，交付量达94.7万辆，而且象征着先进技术与本土制造能力的完美融合。它证明了战略性本土化如何推动全球竞争力。

### 特斯拉德州工厂达到5000万4680电池单元生产里程碑

特斯拉位于德克萨斯州的工厂已生产超过5亿个4680电池单元，足以供应12,000辆Model Y车型。这一里程碑标志着特斯拉在电池生产能力上的重大进步。4680电池单元，以其尺寸（46毫米乘以80毫米）命名，比之前的特斯拉电池更大，承诺提供更高的能量密度和更低的成本。随着特斯拉扩展其电动汽车阵容，包括Cybertruck和新推出的经济型车型，这一进步将至关重要。

近期的发展凸显了特斯拉的积极增长策略：
- 在内华达州新建的电池工厂，装备有领先电池制造商CATL的技术。
- 即将在上海启动的能源存储产品超级工厂，即将开始运营。
- 在墨西哥扩张，建立一个大规模设施以生产多款电动汽车。

这些举措强调了特斯拉扩大生产规模和多样化产品线的承诺，使其在全球电动汽车市场中占据主导地位。

本质上，特斯拉的快速扩张和技术进步，不仅仅关乎制造汽车；它们关乎重塑汽车行业，并加速向可持续能源的过渡。

### 微信支付用户2023年减少碳排放1881.5亿吨

2023年，中国的数字支付平台微信支付用户通过采用电子发票、餐厅数字点餐、在线账单支付和环保旅行选项，显著减少了1881.5亿吨的碳排放。该平台发放了约434亿张电子收据和发票，较上一年惊人地增长了六倍。

转向电子交易，尤其是使用数字收据而非纸质收据，对环境产生了深远影响。每张电子收据都消除了对纸张的需求，减少了森林砍伐以及纸张生产和处理过程中的能源消耗。这种数字化转型不仅简化了财务交易，还促进了绿色经济的发展。

微信支付自2018年起在其电子发票系统中整合了区块链技术，增强了财务记录的透明度和安全性。区块链作为一种去中心化的账本系统，确保每笔交易都被记录且不可更改，提供了可靠的审计跟踪，减少了欺诈的可能性。

微信支付的这一举措凸显了数字技术推动环境可持续性的潜力。通过采用这些工具，用户不仅在管理财务，还在为全球应对气候变化的努力做出贡献。

### DJI的平价驾驶辅助技术套件将革新汽车行业

<img
  src="https://image.informedainews.com/2024/06/06/Te-9yK-1717675972-0.jpg"
  alt="DJI的平价驾驶辅助技术套件将革新汽车行业"
/>

大疆汽车，无人机巨头大疆的子公司，正在革新汽车中驾驶辅助技术的可负担性。通过利用无人机技术，他们以每辆车仅5000元人民币的成本提供2级自动驾驶，使其在售价约10万元人民币的汽车中也能实现。这与华为等竞争对手形成鲜明对比，后者的系统通常出现在更昂贵的车型中。

今年至少有20款新车型将采用大疆的技术，该技术使用摄像头和廉价的半导体，而不是昂贵的激光雷达和雷达系统。这种方法借鉴了大疆Phantom 4无人机技术，使车辆能够根据摄像头输入控制方向和速度。

大疆汽车技术的吸引力在于其成本效益，可能加速驾驶辅助系统的采用。中国优先发展此类技术，目标是到2030年，2级和3级车辆占新车销量的70%，4级车辆占20%。

大疆汽车计划在今年年底前推出一款更先进的系统，能够在常规道路上自动驾驶，每辆车定价7000元人民币。这一举措可能进一步普及自动驾驶技术，使其面向更广泛的市场。

2级自动驾驶指的是系统在某些条件下可以控制转向和加速/减速，要求驾驶员监控环境并准备接管。4级自动驾驶允许车辆在某些条件下无需任何人为干预处理所有驾驶方面。

### 波音公司的星际线飞船成功发射首次载人任务

<img
  src="https://image.informedainews.com/2024/06/06/Te-kUB-1717675976-0.jpg"
  alt="波音公司的星际线飞船成功发射首次载人任务"
/>

2024年6月5日，波音公司的Starliner飞船在经历多次延误后，终于从卡纳维拉尔角成功发射了其首次载人任务。这艘搭载着NASA宇航员巴什·威尔莫和苏尼·威廉姆斯的太空舱，旨在与国际空间站（ISS）对接。此次成功发射对NASA而言意义重大，它为低地球轨道宇航员运输提供了除SpaceX的龙飞船之外的第二种选择。

Starliner，正式名称为CST-100，是一项技术奇迹，其自主飞行系统大大减少了宇航员手动控制的需求。其设计采用无焊结构，通过最小化结构故障风险来增强安全性。太空舱内部宽敞，类似于中型SUV，最多可容纳七人，尽管为ISS任务配置为四人加货物。

此次任务对波音公司至关重要，该公司近期在航空领域面临挑战。Starliner成功融入NASA的商业载人计划可能带来六次额外的宇航员轮换飞行，巩固波音在人类太空飞行中的地位。

两位经验丰富的宇航员对Starliner的开发和准备做出了重大贡献。考虑到太空探索的高风险性质，他们的专业知识尤为宝贵，安全始终是首要考虑。

这次发射不仅推动了太空技术的发展，也突显了数百名NASA和波音员工以及横跨37个州的450家供应商的集体努力。这是人类智慧和对太空探索不懈追求的证明。

### 台積電捐贈40億培育半導體人才，擴大資本預算支持技術發展

<img
  src="https://image.informedainews.com/2024/06/06/Te-AMX-1717675981-0.jpg"
  alt="台積電捐贈40億培育半導體人才，擴大資本預算支持技術發展"
/>

台积电，全球领先的晶圆代工企业，宣布捐赠40亿新台币以培育半导体人才。这笔资金将分配给国立台湾大学、国立阳明交通大学、国立清华大学、国立成功大学及部分高中。此举旨在支持长期的半导体研究和教学，以应对市场对技术人才的需求。

此外，台积电董事会批准了约173亿美元的资本预算，用于扩建和升级其先进制程和封装产能，以及厂房建设。公司还计划增资其子公司TSMC Global Ltd.，以降低外汇避险成本，并从市场买回3,249仟股普通股，以抵消因发行限制员工权利新股而导致的股权稀释。

**解释：**

- **晶圆代工**：指的是专门生产半导体晶片的制造服务，客户提供设计，代工厂负责生产。
- **资本预算**：企业用于长期投资的资金，如购买设备或建设新厂房。
- **先进制程**：指最新的半导体制造技术，能够生产更小、更高效的晶片。
- **外汇避险**：企业为了减少因汇率变动带来的风险，进行的财务操作。
- **限制员工权利新股**：一种股票奖励计划，员工需达到特定条件才能获得股票的所有权。

**洞察：**

台积电的这一系列举措显示了其对未来技术发展的深远布局。通过投资于教育和研发，台积电不仅强化了自己的技术领先地位，也为整个行业的持续创新和人才培养做出了贡献。这种前瞻性的投资策略，对于维持和扩大其在全球半导体市场的领导地位至关重要。

<img
  src="https://image.informedainews.com/2024/06/06/Ar-mUB-1717676008-0.jpg"

<img
  src="https://image.informedainews.com/2024/06/06/Ar-fiA-1717676008-1.jpg"

今日，两架中国制造的飞机——ARJ21和C919——使用了部分由回收食用油（俗称“地沟油”）制成的燃料飞行。这种混合燃料中生物燃料占比40%，标志着国内商业航空领域的一次首创。两架飞机分别从山东和上海起飞，顺利完成飞行任务。

这种生物燃料，被称为可持续航空燃料（SAF），在其生命周期内可减少高达80%的碳排放。它来源广泛，可由废弃油脂、垃圾甚至捕获的二氧化碳制成。在镇海的一家工厂精炼过程中，确保了燃料符合所有标准，与传统喷气燃料无异。

这次飞行不仅仅是技术上的突破，更是向绿色航空产业迈进的一大步。通过将废弃物转化为燃料，中国展示了其创新能力和减少碳足迹的决心。这确实是一个充满希望的开始。

### 江南大学开发出用于合成生物学的高级碱基编辑器

<img
  src="https://image.informedainews.com/2024/06/06/Ar-o3V-1717676017-0.jpg"
  alt="江南大学开发出用于合成生物学的高级碱基编辑器"
/>

江南大学的一个团队开发了一种新型碱基编辑器，将其编辑窗口扩展至41个核苷酸，这一进步是合成生物学中创建高性能工业底盘菌株的关键。研究者通过优化Cas蛋白和融合方法，提高了编辑器的精确度和效率。在E. coli和B. subtilis中的测试表明，基因表达和分泌能力有了显著提升。

**关键概念解释：**

- **碱基编辑器：** 一种直接改变DNA中单个核苷酸而不切割DNA链的工具，降低了产生不必要突变的风险。
- **底盘菌株：** 在工业生物技术中用作生产各种化合物基础的工程化微生物细胞。
- **CRISPR-Cas：** 一种基因编辑技术，利用Cas蛋白精确靶向和修改特定的DNA序列。

**洞察：**

这一突破不仅提高了基因编辑的精确性，还加速了更高效生物生产系统的发展。通过实现更广泛和精确的遗传修改，这项技术可能彻底改变依赖微生物合成的行业，从制药到生物燃料。人工智能在设计和优化这些工具中的整合进一步强调了将计算方法与生物研究相结合的变革潜力，为合成生物学的新时代铺平了道路。

### Ubuntu Core 24 发布：针对物联网和嵌入式设备优化

<img
  src="https://image.informedainews.com/2024/06/06/Ar-pbq-1717676017-0.jpg"
  alt="Ubuntu Core 24 发布：针对物联网和嵌入式设备优化"
/>

Canonical 发布了 Ubuntu Core 24，这是针对物联网和嵌入式设备定制的 Ubuntu 24.04 LTS 完全容器化版本。该版本支持广泛的硬件，包括 Raspberry Pi、Qualcomm DragonBoard、Intel NUC，以及基于 x86_64 和 RISC-V 架构的设备。

Ubuntu Core 24 引入了完全可抢占的内核，增强了响应能力，并在认证硬件上启用了先进的实时功能。关键特性包括可定制的图像验证集、增强的自动网络请求控制，以及为 AIoT 设备和图形应用改进的 GPU 集成。它还支持 Ubuntu Frame 以增强嵌入式 Linux 安全性，并集成 Canonical Landscape 进行设备管理。

此版本与 Azure IoT Edge 无缝集成，并支持机器人操作系统（ROS）。Ubuntu Core 24 本质上是一个为下一代智能设备，特别是 AIoT 和边缘计算领域设计的强化嵌入式 Ubuntu 版本。它提供了一个完整的 AI 堆栈，包括实时内核，这对于在工业、汽车和安全敏感的嵌入式解决方案中做出关键决策至关重要。

最低系统要求为 384MB RAM 和 512MB 存储，UEFI 安全启动和 FED 则需要更高的要求。该版本支持众多设备，确保了广泛的应用性。

新研究揭示，来自澳大利亚的古代锆石晶体表明，地球在40亿年前就拥有了淡水和陆地，这比先前认为的早了5亿年。科廷大学的科学家们分析了来自西澳大利亚州杰克山的这些已知最古老的陆地矿物锆石晶体。锆石因其极高的抗风化和转化能力，被誉为“时间之主”，是可靠的历史记录者。

研究团队对大约1000颗锆石颗粒进行了年代测定，发现其中10%超过40亿年。通过测量这些晶体中轻重氧同位素的比例，他们检测到了一种独特的轻氧特征，表明存在淡水。这一发现表明，对维持生命至关重要的水文循环至少始于40亿年前。

这一证据挑战了先前关于地球在30亿至40亿年前完全被海洋覆盖的理论。这些发现不仅照亮了地球早期历史，还暗示陆地和淡水可能促进了地球形成后6亿年内生命的迅速出现。

**关键术语解释：**
- **锆石：** 一种常用于岩石年代测定的矿物，因其耐久性和含有放射性同位素而著称。
- **同位素：** 同一元素的不同变体，具有相同数量的质子但中子数量不同。
- **水文循环：** 水在地球表面、上方和下方持续运动的过程。
