Canonical 发布了 Ubuntu Core 24，这是针对物联网和嵌入式设备定制的 Ubuntu 24.04 LTS 完全容器化版本。该版本支持广泛的硬件，包括 Raspberry Pi、Qualcomm DragonBoard、Intel NUC，以及基于 x86_64 和 RISC-V 架构的设备。

Ubuntu Core 24 引入了完全可抢占的内核，增强了响应能力，并在认证硬件上启用了先进的实时功能。关键特性包括可定制的图像验证集、增强的自动网络请求控制，以及为 AIoT 设备和图形应用改进的 GPU 集成。它还支持 Ubuntu Frame 以增强嵌入式 Linux 安全性，并集成 Canonical Landscape 进行设备管理。

此版本与 Azure IoT Edge 无缝集成，并支持机器人操作系统（ROS）。Ubuntu Core 24 本质上是一个为下一代智能设备，特别是 AIoT 和边缘计算领域设计的强化嵌入式 Ubuntu 版本。它提供了一个完整的 AI 堆栈，包括实时内核，这对于在工业、汽车和安全敏感的嵌入式解决方案中做出关键决策至关重要。

最低系统要求为 384MB RAM 和 512MB 存储，UEFI 安全启动和 FED 则需要更高的要求。该版本支持众多设备，确保了广泛的应用性。
