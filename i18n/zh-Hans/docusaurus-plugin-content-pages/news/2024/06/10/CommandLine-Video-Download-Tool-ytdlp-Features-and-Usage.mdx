# yt-dlp 命令行视频下载工具：功能与使用

yt-dlp 是一个功能强大的命令行程序，用于从互联网上下载视频，特别是从 YouTube 网站。它是基于 youtube-dl 的一个分支，增加了额外的功能和改进。以下是 yt-dlp 的一些主要功能和使用方法：

功能：
1. 支持从多个视频平台下载视频，包括 YouTube、Vimeo、Dailymotion 等。
2. 可以选择下载视频的不同格式和质量，包括音频和视频的单独下载。
3. 支持下载播放列表和频道中的所有视频。
4. 可以提取视频的字幕，并支持多种字幕格式。
5. 支持通过代理服务器下载视频。
6. 提供丰富的命令行选项，如设置下载速度限制、自动重试下载等。

使用方法：
1. 安装 yt-dlp：首先需要在计算机上安装 yt-dlp。可以通过 pip 安装，命令为 `pip install yt-dlp`。
2. 基本使用：在命令行中输入 `yt-dlp [视频链接]`，即可开始下载视频。
3. 选择视频格式：可以使用 `-f` 参数指定下载的视频格式，例如 `yt-dlp -f bestvideo+bestaudio [视频链接]` 下载最佳视频和音频组合。
4. 下载播放列表：使用 `yt-dlp [播放列表链接]` 可以下载整个播放列表中的视频。
5. 提取字幕：通过添加 `--write-sub` 和 `--write-auto-sub` 参数，可以下载视频的字幕和自动生成的字幕。
6. 使用代理：如果需要通过代理下载，可以使用 `--proxy` 参数指定代理服务器的地址。

yt-dlp 是一个非常灵活和强大的工具，适合需要从互联网下载视频的用户。通过掌握其命令行选项，用户可以更有效地管理和控制下载过程。
